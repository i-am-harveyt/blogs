# Introduction

我們先從簡單的開始：在本地端使用 Git。

接下來的範例都會是 UNIX 環境下的指令。
不知道什麼是 UNIX 環境也沒關係，只要你用的是 git bash, bash, zsh 等等就行。

接下來的所有操作都會在 `~/Desktop/example/` 路徑完成，簡單來說就是在桌面上的 example 資料夾，對終端機不熟的話可以手動用滑鼠右鍵在桌面上建立。

這邊給一個指令：

```bash
$ mkdir ~/Desktop/example && cd ~/Desktop/example
```

如果不確定有沒有成功的話可以下指令 `pwd` 確認一下自己的路徑。

接下來我會階段性的提供視覺化的圖，那是用 [Mermaid JS](https://mermaid.js.org/) 寫出來的。
