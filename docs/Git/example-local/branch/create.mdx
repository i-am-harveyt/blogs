# Create New Branch

來新增一個 branch 吧！今天我們來豐富我們的文件，給他一個 "Introduction"

```bash
$ git branch doc/introduction
```

我們來看一下我們的 branches with `git branch`

```bash
$ git branch
  doc/introduction
* main
```

可以看到現在除了 `main` branch 以外多出了 `doc/introduction` branch，並且我們現在在 main branch 上工作。
