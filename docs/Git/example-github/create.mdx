# Create a New Repo

你可以從你的主頁找到一個綠色的 "New" 按鈕，或點擊[這裡](https://github.com/new)。

按照圖片所示設定。因為我已經創建過同名 repo 所以會報錯，你應該不會有這個問題。正好可以對比一下如果名字重複會有什麼差別。

![image](https://hackmd.io/_uploads/HJOAX3jwJx.png)
